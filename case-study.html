Liquid State | A Responsive Theme

About

Inspired by HTML5's Solid State Theme with a creative twist. The sections get Darker as you scroll down, as a parallel to a body of water getting darker the deeper you travel. Furtering the nautical element, I used waves as dividers between the Hero Section and the rest of the Web Page.

Key Features

Mobile First

Responsive

Parallax Backgrounds

SVG Animations

Technologies Used

Hosted via Github Pages
